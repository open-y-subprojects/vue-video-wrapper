# Vue Video Wrapper
A Vue.js component to wrap embed iframes from Vimeo and Youtube.

### Intro

A simple Vue.js component where you can wrap your **Vimeo** or **Youtube** embeded video and use their events.

### Installation

Using npm:

    npm install vue-video-wrapper
